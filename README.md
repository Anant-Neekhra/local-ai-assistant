# Local AI Assistant with Ollama

## Overview

This project implements a local AI assistant using Small Language Models (SLMs) running completely offline with Ollama.

The system benchmarks models and compares performance across different configurations.

## Models

- Llama 3.2
- Mistral 7B
- Phi

## Features

- Local AI assistant
- Offline inference
- Model benchmarking
- Structured outputs
- Model comparison

## Setup

Install Ollama:

https://ollama.com

Pull models:

ollama pull llama3.2
ollama pull mistral
ollama pull phi

Run assistant:

python assistant.py
